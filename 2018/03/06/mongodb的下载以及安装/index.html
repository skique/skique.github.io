mongodb的下载以及安装

下载mongodb

  • 下载地址为:https://gitee.com/taoyouyou/mongodb-install.git,此版本只适用于window 64位系统。

    安装：

1、在安装过程中可以点击Custom(自定义)” 按钮来设置安装目录。我的安装目录为 D:\MongoDB\Server\3.4\bin，之后直接点击下一步即可安装成功。

2、安装mongodb 后设置mongodb环境变量（如果没有配置环境变量，一下所有命令均在mongodb安装目录的bin目录下执行）

3、新建数据库文件夹和日志记录文件

在D:\MongoDB\Server\3.4目录下新建data文件夹作为数据库路径,新建logs文件夹下的mongo.log文件作为日志文件以及新建mongo.conf配置文件

4、数据库配置文件
在mongo.conf中写入以下配置信息

5、启动mongodb服务

  • 方法一
    mongod --config "D:\MongoDB\Server\3.4\mongo.conf" // 在命令行启动
  • 方法二
    mongod --config "D:\MongoDB\Server\3.4\mongo.conf" --install --serviceName "MongoDB" // 安装MongoDB的windows服务管理

右键计算机——系统服务，打开windows服务框，本地服务服务列表中会看到MongoDB，表示已经成功将mongodb加入到了系统服务，但是此时服务还没有开启！

6、创建mongodb连接
安装完成后，shell交互式下输入mongo就可以直接无密码登录到数据库。默认登录到test数据
mongo 127.0.0.1:27017/admin -u 'userAdmin' -p '123' // 使用用户名和密码连接到MongoDB服务器
